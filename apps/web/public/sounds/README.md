# 🔊 KYC Voice Guide — Audio Assets

Đặt các file MP3 sau vào thư mục này:

| File             | Khi nào phát                              |
|------------------|-------------------------------------------|
| `left.mp3`       | Challenge: quay đầu sang trái             |
| `right.mp3`      | Challenge: quay đầu sang phải             |
| `on.mp3`         | Challenge: ngước đầu lên                  |
| `below.mp3`      | Challenge: cúi đầu xuống                  |
| `keep-the-same.mp3` | Khi user vào đúng tư thế (giữ nguyên) |

> **Lưu ý:** Nếu user đang giữ đúng tư thế mà di chuyển sai → hệ thống sẽ tự phát lại voice của hành động đó (left, right, on, below...).
